01. Project Overview
Background
JAPA is a free mobile guidance app for consumers that provides access to accurate real-time parking availability, parking lot information, and policies. People have places to be and JAPA helps them get there faster. The free app makes it easy to find parking without driving around racing to a space. Users can view spots ahead of time to plan out their parking experience. JAPA allows users to park smarter and stress less when they're on the go.
Objective
Currently, JAPA‚Äôs mobile architecture lacks a well-defined workflow, making it slow and inefficient. Our solution is to design a unique, intuitive mobile application for finding parking that will provide a seamless experience that saves time and money.
Current Mobile App
These first three screens make up the main menu of JAPA's mobile app. (Left to right) We have the map screen, which is the first screen users see upon opening the app. It displays green parking pins signaling where the closest parking lots are relative to the users current location.

Next is the "Find Parking" swipe up tab, which displays all the parking lots that are compatible with the JAPA app.

Lastly is the slide out menu tab, which allows users to find out more information, report any issues, and clarify any questions.
